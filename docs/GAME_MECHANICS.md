# Cognitive Crucible Game Mechanics Summary

This document provides a concise overview of the core gameplay mechanic for each of the 8 primary CHC domain training games in their default "Core" mode.

---

### 1. (Gf) Fluid Reasoning - **Pattern Matrix**

-   **Objective:** Deduce underlying abstract rules.
-   **Mechanic:** You are shown a grid of symbols (e.g., 3x3) with one piece missing. The shapes, colors, or rotations of the symbols follow a logical pattern along the rows and columns. You must analyze the pattern and select the correct missing piece from a set of options.

---

### 2. (Gc) Crystallized Intelligence - **Verbal Inference Builder**

-   **Objective:** Apply learned knowledge and vocabulary.
-   **Mechanic:** You are presented with a verbal puzzle, typically a sentence with a missing word (a "cloze deletion" task). You must use your vocabulary and understanding of the context to choose the word that best completes the sentence from a list of options.

---

### 3. (Gwm) Working Memory - **Dynamic Sequence Transformer**

-   **Objective:** Hold and manipulate information in your active memory.
-   **Mechanic:** A short sequence of letters and/or numbers is displayed on the screen for a few seconds. After it disappears, you are given a rule (e.g., "Reverse the sequence"). You must then type the correctly transformed sequence from memory.

---

### 4. (Gs) Processing Speed - **Rapid Code Match**

-   **Objective:** Perform a simple cognitive task as quickly and accurately as possible.
-   **Mechanic:** A key is shown at the top, mapping abstract symbols to numbers (e.g., ★ -> 1, ▲ -> 2). A single symbol then appears in the center of the screen. You must press the corresponding number key as fast as you can. The key mapping changes periodically to prevent motor memorization.

---

### 5. (Gv) Visual Processing - **Mental Rotation Lab**

-   **Objective:** Mentally manipulate 2D shapes.
-   **Mechanic:** You are shown a single "target" shape made of blocks. Below it are several "option" shapes. You must identify which of the options is an identical, but rotated, version of the target shape. Mirrored versions are distractors and must be ignored.

---

### 6. (Ga) Auditory Processing - **Auditory Processing Lab**

-   **Objective:** Analyze and distinguish different qualities of sound.
-   **Mechanic:** This is a "lab" or a hub of several mini-games. By default, you might play:
    -   **Pitch Discrimination:** You hear two tones and must decide if the second is higher or lower than the first.
    -   **Timbre Identification:** You hear a single note played by an instrument and must identify which instrument it was (e.g., Piano, Bell).
    -   **Sound Localization:** You hear a sound and must identify its position in stereo space (Left, Center, or Right).

---

### 7. (Glr) Long-Term Retrieval - **Retrieval Trainer**

-   **Objective:** Efficiently access information stored in your long-term memory.
-   **Mechanic:** The default mode is **Category Switching**. You are given a category (e.g., "Animals," "Fruits") and a time limit. You must type as many unique words belonging to that category as you can. The category will switch automatically, forcing you to shift your mental search.

---

### 8. (EF) Executive Function - **Focus Switch Reactor**

-   **Objective:** Maintain focus and switch between conflicting rule sets.
-   **Mechanic:** A stimulus (e.g., a colored shape) appears on screen. A rule is also displayed, such as "Respond to the SHAPE" or "Respond to the COLOR." You must respond based on the current rule using arrow keys. The rule changes periodically, forcing you to inhibit the old rule and apply the new one.
